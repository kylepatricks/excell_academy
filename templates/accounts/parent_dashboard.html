How to View Report Cards
Report cards are available after they are generated by teachers.
To view report cards:
  1. Click on "View All Report Cards" to see all available reports
  2. If a report card has a "Preview" button, you can view it in your browser
  3. If a report card has a "Download" button, you can save a PDF copy
  4. Report cards are typically generated at the end of each term

If you don't see any report cards, please contact your child's teacher or the school administration.
